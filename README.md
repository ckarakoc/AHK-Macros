## volume.ahk

## special_characters.ahk

## windows.ahk
[Superkey] + [f] = firefox <br />
[Superkey] + [c] = chrome <br />
[Superkey] + [SHIFT] + [f] = firefox private window <br />
[Superkey] + [SHIFT] + [c] = chrome private window
